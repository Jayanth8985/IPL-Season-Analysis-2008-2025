# 🏏 IPL Season Analysis (2008–2025)

## 📌 Project Overview
End-to-end data analysis of all IPL seasons from 2008 to 2025
using Microsoft Excel covering 18 seasons of match data.

⚠️ **Important Note**

After downloading the Excel file from GitHub, please click **"Enable Editing"** at the top of Excel to interact with slicers, pivot tables, and dashboard visuals.

## 🔍 Analysis Performed
- Toss Decision Analysis — 81% of toss winners chose to field first
- Team-wise Win Performance using Pivot Tables
- Top 10 Venue Analysis by match frequency
- Man of the Match award frequency analysis
- Historical Winners Table (2008–2025)
- KPI Dashboard — Season Winner, Runner-up, Player of the Season

## 🛠️ Tools Used
Microsoft Excel — Pivot Tables, Charts, Slicers, KPI Cards, Dashboard

## 🏆 Key Insights
- CSK & MI — joint all-time leaders with 5 titles each
- RCB won IPL 2025 | Player of the Season: Suryakumar Yadav
- Narendra Modi Stadium hosted the most matches
